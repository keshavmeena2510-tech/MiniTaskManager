\# MiniTaskManager



This is a simple Task Management project made using Java, JSP, Servlets, and JDBC.



\## Features

\- Add Task

\- View Task

\- Update Task

\- Delete Task



\## Tech Stack

\- Java

\- JSP \& Servlets

\- JDBC

\- SQL Database

\- Apache Tomcat



\## How to Run

1\. Import project in NetBeans/Eclipse/IntelliJ.

2\. Setup database.

3\. Run on Apache Tomcat.

4\. Open in browser: http://localhost:8080/MiniTaskManager
